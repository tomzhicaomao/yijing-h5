# yijing-h5 路线图（重建版）

## 当前定位
- 项目：易经占卜 H5
- 目标：稳定、可维护、可持续迭代
- 技术：Vite + 原生前端模块化（逐步演进）

## 里程碑

### M1 架构清理（1周）
- 拆分 `index.html` 内联脚本与样式
- 建立模块结构（`src/modules/*`）
- 统一构建与运行命令
- 完成基础 lint/format

验收标准：
- `npm run dev` 可运行
- `npm run build` 可产出
- 页面核心流程不回退

### M2 核心体验优化（1周）
- 修复移动端适配
- 优化起卦流程与状态反馈
- 完善历史记录展示（本地）

验收标准：
- 核心路径可用（起卦/查看结果/查看历史）
- 移动端主流分辨率可正常使用

### M3 后端能力接入（1周）
- 集成 Supabase Auth（官方流程）
- 接入历史记录云端同步
- 增加环境变量与部署配置

验收标准：
- 登录/登出可用
- 历史记录可云端读写
- 生产环境可部署

### M4 质量与发布（持续）
- 增加自动化测试（单元 + 关键E2E）
- 增加性能与异常监控
- 完成发布与回滚流程固化

验收标准：
- QA 无 P0/P1
- 发布可回滚

## 非目标（当前阶段）
- 不做多端 App 打包
- 不做复杂推荐算法
- 不引入重型后端框架
